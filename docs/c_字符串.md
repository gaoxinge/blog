下面我们研究一下c中的字符串。

## char数组

```c
char a[] = "1234";
char a[] = {'1', '2', '3', '4', '\0'};
```

## 指针

```c
char *a = "1234";
```

## 区别

- 字符串常量属于静态存储（static storage）类，被引用的字符串位于静态存储区
- 数组初始化是从静态存储区把一个字符串复制给数组
- 指针初始化是复制字符串的地址

```c
#include <stdio.h>

int main()
{
  printf("%p", *"1234"); // 其他常量不行
  return 0;
}
```
