﻿python如此受欢迎的一个重要原因是强大的第三方库支持。

管理

Github上的pypa提供了一个有关包的指南Python Packaging User Guide。另外我们可以通过官网的PyPI查看第三方包的信息。其中许多包是托管在Github上的，比如Kenneth Reitz下的许多仓库。

包的管理基本分为以下八个步骤：

  • 打包
  • 发布
  • 下载
  • 安装
  • 升级
  • 卸载
  • 查看
  • 使用

工具

python有许多包管理工具，常用的有distutils，setuptools和pip，另外Python包管理工具解惑解释了管理工具间的区别。

安装

具体可参见windows下面安装Python和pip终极教程。

distutils和setuptools

  • 文档
  • 文档
  • 常用命令
# 打包
python setup.py sdist         # tar.gz
python setup.py bdist_egg     # egg
python setup.py bdist_rpm     # rpm
python setup.py bsist_wininst # exe

# 安装
python setup.py install

pip

  • 文档
  • 常用命令
# 下载，安装，升级和卸载
pip install requests           # 下载和安装
pip install --upgrade requests # 升级
pip uninstall requests         # 卸载 

# 查看
pip list
pip list --outdated
